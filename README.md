# AI Productivity Intelligence

## Problem
Many students feel stressed and unproductive but don't know why. 
This project aims to analyze basic daily habits like study hours and sleep 
to give simple productivity and stress insights.

## Goal
To build a real-world AI-based system that improves my resume 
and helps me secure an internship.

## Planned Tech Stack
- Python
- Pandas, NumPy
- Machine Learning (later)
- FastAPI (later)
- React (later)
